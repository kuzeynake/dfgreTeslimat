# dfgreTeslimat
# Teslimat Sistemi

Minecraft sunucuları için oyuncuların belirli eşyaları teslim ederek rekabet etmesini sağlayan bir sıralama ve ödül sistemidir.

## Özellikler

* **Eşya Teslimatı:** Belirlenen hedef eşyaları (havuç vb.) sunucuya teslim ederek puan kazanırsınız.
* **Canlı Sıralama:** `/teslimat` komutu ile ilk 10'a giren oyuncuları anlık olarak görebilirsiniz.
* **Dinamik Ödüllendirme:** Teslimat süreci sonunda ilk 3 sıradaki oyunculara  ödül verilir.
* **GPLv3 Lisanslı:** Kodlar her zaman açık ve geliştirilebilir kalacaktır.

## Kurulum

1. `releases` sekmesinden `.jar` dosyasını indirin.
2. Dosyayı sunucunuzun `plugins` klasörüne taşıyın.
3. Sunucuyu başlatın.

## Kullanım ve Komutlar

| Komut | Açıklama |
| :--- | :--- |
| `/teslimat` | Teslimat menüsünü açar. |
